# TRACE Comparator Standard v0.1

Date: 2026-06-17
Status: comparator rule / not canon / not validation

## Plain purpose

This file corrects the standard used to judge TRACE against existing fields.

TRACE is allowed to function as a unifying language. It does not have to invent every component it uses. Existing fields may already contain the same move under local vocabulary. That is not automatically failure.

The question is whether TRACE preserves, compresses, and transfers the move without loss.

```trace
TRACE_comparator_standard :=
  not_novelty_only
  but:
    unification
    + compression
    + translation
    + transfer
    + action_routing
    + loss_detection
    + demotability
```

## Wrong test

```trace
wrong_test :=
  existing_field_already_contains_component
  -> TRACE_demoted
```

This is too strong. It would kill any unifying system. A universal language is expected to translate existing knowledge, not replace every field.

## Right test

```trace
right_test :=
  does_TRACE_translate_across_domains
  while:
    preserving_relevant_structure
    + reducing_navigation_cost
    + exposing_loss
    + keeping_demoters_live
```

Plain version: TRACE is useful if it lets a reader move between law, AI, infrastructure, ecology, bodies, economics, belief, and systems without losing the structure that matters.

## Component novelty vs structural delta

TRACE can add value in several ways.

```trace
TRACE_value_modes :=
  component_novelty
  OR structural_compression
  OR cross_domain_translation
  OR faster_navigation
  OR better_loss_detection
  OR clearer_action_routing
  OR demotable_unification
```

Component novelty is only one value mode. It is not the master test.

## What counts as a real delta

```trace
real_delta_if:
  TRACE_makes_visible
    a_cross_domain_structure
  AND:
    ordinary_local_field_does_not_route_it_across_domains_as_cleanly
  AND:
    the_translation_preserves_case_specific_detail
```

A TRACE delta may be a cross-domain translation even when every component has a known analogue in another discipline.

## What does not count

```trace
no_delta_if:
  TRACE_only_renames
  OR adds_vocabulary_without_navigation_gain
  OR hides_local_expertise
  OR weakens_source_specificity
  OR makes_cases_look_more_similar_than_they_are
```

TRACE fails when its universal language flattens local knowledge or creates false equivalence.

## Demotion standard

The demotion standard should change from pure novelty to net navigational value.

```trace
demote_if:
  no_cross_domain_compression
  + no_navigation_gain
  + no_loss_detection
  + no_action_routing_gain
  + more_vocabulary_than_value
```

Existing literature already containing the component is not enough for demotion. Existing literature containing the component with equal transfer, equal clarity, equal action-routing, and less burden is enough for demotion.

## Comparator pass categories

When running a source-backed comparator, classify each TRACE component into one of five states.

```trace
component_status :=
  captured_locally
  OR compressed_by_TRACE
  OR transferred_by_TRACE
  OR improved_by_TRACE
  OR flattened_by_TRACE
```

Definitions:

- `captured_locally`: existing field already handles the move well inside its domain.
- `compressed_by_TRACE`: TRACE gives a shorter or clearer route to the same structure.
- `transferred_by_TRACE`: TRACE lets the move travel into other domains without losing key structure.
- `improved_by_TRACE`: TRACE catches a timing, gate, subject, or repair feature the local field under-emphasises.
- `flattened_by_TRACE`: TRACE makes the field worse by erasing specific detail.

## Source respect rule

```trace
source_respect_rule :=
  local_field_keeps_priority
  for:
    factual_detail
    + technical_method
    + legal_doctrine
    + domain_specific_evidence
```

TRACE may translate. It must not pretend to be a substitute for the source field.

## Universal-language guardrail

```trace
universal_language_valid_if:
  local_meaning_preserved
  + cross_domain_mapping_clear
  + demotion_allowed
  + no_totalising_claim
```

A universal language becomes dangerous when it treats translation as ownership.

```trace
universal_language_failure :=
  translation_as_ownership
  OR analogy_as_equivalence
  OR compression_as_proof
  OR scaffold_as_result
```

## Applying this to current worked-delta cases

### Apollo 13

```trace
Apollo_status_under_new_standard :=
  local_field_strong
  + TRACE_compression_possible
  + transfer_value_possible
  but:
    engineering_specificity_must_remain_primary
    + success_case_limits_delta
```

Apollo 13 should not be demoted merely because high-reliability and safety-engineering fields already contain many components. It should be demoted only if TRACE adds no useful cross-domain compression beyond those fields.

### Prediction-authority / Minority Report

```trace
Prediction_Authority_status_under_new_standard :=
  due_process_and_algorithmic_accountability_capture_much
  + TRACE_may_compress_timing_gate_future_space
  but:
    open_future_and_scored_society_comparators_still_needed
```

The surviving candidate is not that TRACE invented contestability, opacity, or due process. It is that TRACE may compress them into a cross-domain timing question:

```trace
can_contestability_reach_the_gate
before_future_space_hardens?
```

## Required wording change in future files

Avoid this unsupported phrasing:

```trace
bad_wording :=
  TRACE_adds_nothing_if_existing_field_mentions_it
```

Prefer:

```trace
good_wording :=
  existing_field_captures_component_locally
  while:
    TRACE_delta_depends_on_cross_domain_transfer
    + navigation_gain
    + loss_detection
```

## Final rule

```trace
final_rule :=
  TRACE_is_not_validated_by_unification
  but:
    TRACE_is_allowed_to_seek_unification

proof_required_for:
  predictive_claims
  factual_claims
  case_claims
  policy_claims

not_required_for:
  using_TRACE_as_translation_interface
  if:
    translation_is_demotable
    + source_fields_remain_primary
```

Plain version:

TRACE can be a universal interface. That is legitimate. The audit standard must not demand new invention at every node. But TRACE only earns the universal-language claim if it translates across domains without flattening, hides nothing, keeps source fields primary, and remains demotable.
