## 3. Main weakness

Claude's strongest criticism:

```text
The I slot conflates invariant and measure.
```

Problem:

```text
An invariant and a measure are not the same type of object.
```

Examples:

```text
topology: Betti numbers are partial invariants, not complete equivalence tests
QIT: fidelity and logical error rate are measures/readouts
epidemiology: R_t is a varying measure, not a class-determining invariant
DSGE: welfare/output/inflation/debt-service indicators are measures or model variables
```

Risk:

```text
same invariant/measure value may be misread as same native class or same system state
```
