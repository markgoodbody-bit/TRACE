# TRACE Cross-Domain Language Coverage Map v0.1

Status: coverage map / language-transfer test.  
Source seed: Qwen eight-domain spread supplied by Mark.  
Not: canon promotion, proof of universality, domain replacement, expert outperformance claim, or new glyph/operator layer.

## 0. Purpose

This file corrects a drift in the TRACE test frame.

TRACE is not currently trying to prove that it is better than every specialist framework. The stronger project claim is different:

```text
TRACE should function as a portable structural language across domains with radically different native mathematics, objects, clocks, constraints, and vocabularies.
```

A good TRACE translation does not replace the native domain.

It should let people locate comparable structural pressures across domains while preserving the fact that each domain has its own machinery.

## 1. Source seed

The seed set contains eight domains chosen precisely because their native mathematics and terminology are far apart:

```text
1 quantum information theory
2 algebraic topology
3 dynamic macroeconomics / DSGE modelling
4 generative syntax / minimalist linguistics
5 epidemiology and compartmental modelling
6 plate tectonics and geodynamics
7 organic synthesis and retrosynthetic analysis
8 neurophysiology and computational neuroscience
```

The reason this set is useful is not that TRACE can derive their equations.

It is useful because it tests whether TRACE can preserve stable structural questions across incompatible native languages.

## 2. Success condition

TRACE succeeds as a bridge language if it can map each domain through stable questions without flattening the domain:

```text
What is the state?
What changes the state?
What constraints shape the change?
What clock matters?
What evidence makes the state knowable?
What failure mode hardens?
What correction channel exists?
What does the native field already call this?
Where does TRACE translation break or become metaphor only?
```

TRACE fails if it:

```text
replaces native mathematics with slogans
pretends structural analogy is proof
turns all domains into ethics cases
smuggles moral subjects into non-subject domains
claims superiority over expert methods
uses glyphs where plain language is clearer
compresses away the domain-specific mechanism
```

## 4. Cross-domain TRACE primitives under test

This map tests whether the following TRACE primitives remain stable across domains:

```text
state
transition
constraint
clock
evidence
threshold
hardening
correction
burden / cost landing
standing / subject boundary
answerability
```

Not every primitive applies equally in every domain.

The important result is not universal application.

The important result is honest applicability marking:

```text
applies cleanly
applies by analogy only
applies only when human/animal/AI subjects enter the system
does not apply
```

## 5. Stable cross-domain patterns

### 5.1 State under constraint

Across all eight domains, there is some state-space-like object:

```text
quantum state
topological space
economic state
syntactic derivation
population compartment state
tectonic stress state
molecular reaction state
neuronal voltage/circuit state
```

TRACE can ask:

```text
what state is being changed, by what rule, under what constraint?
```

This is a language bridge, not a replacement for native state mathematics.

### 5.2 Clock mismatch

Many domains have failure when one clock outruns another:

```text
decoherence outruns error correction
intervention delay outruns epidemic spread
recovery pressure outruns review
reaction side-path outruns desired selectivity
seismic strain accumulates beyond ordinary observation cycles
synaptic/plasticity change outruns regulatory control
```

TRACE language:

```text
harm_or_failure_clock > correction_or_control_clock
```

Risk:

```text
clock-language becomes too general unless tied to a native mechanism.
```

### 5.3 Boundary / accessibility

Several domains include reachability or access limits:

```text
phase boundary in syntax
measurement limits in quantum theory
protected functional group in synthesis
network edge/contact structure in epidemiology
plate boundary in geodynamics
synaptic/circuit topology in neuroscience
institutional review route in governance cases
```

TRACE language:

```text
what can still be reached, altered, protected, or corrected from here?
```

Risk:

```text
boundary does not always mean ethical boundary.
```

### 5.4 Irreversibility / hardening

Domains differ sharply in what counts as irreversible:

```text
quantum measurement/decoherence may make phase relation unrecoverable in practice
topological tearing changes equivalence class
macroeconomic shock may create path dependence
syntax phase transfer makes internal structure less accessible
infection spread creates downstream cases
tectonic rupture releases accumulated strain
chemical side reaction consumes material
neural plasticity can stabilise maladaptive pathways
```

TRACE language:

```text
when does later correction stop being restoration and become only compensation / record / adaptation?
```

Risk:

```text
not every irreversible change is harm.
```

### 5.5 Correction channel

Each field has some analogue of correction, but the meaning differs:

```text
quantum error correction
homology invariant correction / proof refinement
macroeconomic policy correction
syntactic reanalysis / grammar refinement
epidemiological intervention
hazard mitigation
synthetic route redesign
homeostatic or clinical intervention
```

TRACE language:

```text
what channel can still alter the trajectory before the failure mode hardens?
```

Risk:

```text
correction is not always moral repair; often it is technical control.
```

## 6. Domain translation discipline

For each domain, TRACE must preserve three layers:

```text
native layer := the field's own mathematics and terms
TRACE layer := shared structural translation
ethics/governance layer := only where subjects, harm, power, or answerability enter
```

Example:

```text
Organic synthesis native layer: reaction kinetics, selectivity, protecting groups.
TRACE layer: path-dependent transformation under constraints with irreversible side-routes.
Ethics layer: only if the molecule, process, supply chain, drug, toxicity, environment, or affected subjects enter the case.
```

This avoids turning TRACE into a metaphor machine.

## 8. Falsifiers

Mark `TRACE_LANGUAGE_COVERAGE_LOW` if most domain translations reduce to vague metaphors such as "everything changes" or "systems need correction."

Mark `TRACE_DOMAIN_FLATTENING_FAILURE` if the translation erases native mechanisms or makes a chemist, physicist, linguist, epidemiologist, geophysicist, neuroscientist, economist, or topologist say: "that is not what my field is doing."

Mark `TRACE_ETHICS_OVERREACH_FAILURE` if non-subject domains are treated as moral subjects without a downstream subject/harm bridge.

Mark `TRACE_LANGUAGE_COVERAGE_PARTIAL` if the structural columns are useful but require heavy domain-specific caveats.

Mark `TRACE_LANGUAGE_COVERAGE_STRONG` only if external domain readers can locate precise agreement/disagreement faster because of the TRACE layer.

## 9. Next tests

Next useful work is not another superiority comparator.

It is a translation stress test:

```text
Pick one native problem from each domain.
Map it through TRACE.
Ask whether TRACE preserves:
  native object
  native clock
  native constraint
  native failure mode
  native correction channel
  boundary between technical failure and moral/governance relevance
```

