```
Core vocabulary, grammar, operators, uncertainty, future-space, method, and bootstrap
anchors. Author: Mark Goodbody with Framework. Date: 15 June 2026.
0. Use Contract
TRACE is a structural language for middle-out reasoning. It is not a magic language, not a proof engine,
and not a substitute for evidence. It must keep uncertainty inside the model. A valid TRACE expression
maps operators to real relations: transition, constraint, timing, memory, correction, uncertainty,
propagation, method, and affected-subject modelling.
truth_rule :=
model != measurement
compression != proof
structure != causation
emergency != permission_to_delete_constraint
equation_valid iff operator_encodes_relation
1. Core Ontology
Symbol / alias
Definition
Omega / World
Active environment: entities, relations, resources, constraints, histories.
E / Entity
Persistent pattern under transition; not static substance.
x_t / State
State of an entity or system at time t.
S / State space
Possible states under the current model.
o_t / Observation
Local slice of world available to an entity.
M / Memory
Bridge from past states to present selection.
T(M) / Trust
Reliability assigned to a memory or record.
C / Constraint
Limit on admissible actions, methods, transitions, or states.
u_t / Action
Selected intervention/output.
m_t / Method
How action is applied; method changes the system state.
pi / Policy
Selection rule choosing action from available options.
tau / Clock
Time-to-event: action, correction, hardening, decay, explosion, etc.
K / Correction
Effective detection, contest, interruption, repair, enforcement.
UQ / Uncertainty
Unknowns, confidence, source quality, scope, unresolved category.
Xi / Future-space
Accessible future option-space for entities/systems.

2. Minimal Middle-Out Grammar
TRACE_LOOP(E, Omega):
1. observe local slice o_t
2. update x_t and UQ_t
3. select trusted memory M_eff
4. model affected entities where relevant
5. restrict actions and methods by C_t
6. choose u_t = pi(x_t, o_t, M_eff, C_t, UQ_t)
7. apply method m_t and action u_t before tau_harden if needed
8. record event and result
9. verify prediction vs observation
10. correct/repair if divergence detected
11. update Rosetta only if a real structural gap is found
3. Identity, Memory, Continuity
entity := persistent_pattern_under_transition
identity := continuity_under_transition + trusted_memory
continuity_error_t = distance(x_{t+1}, predict(x_t, M_eff, f_hat))
identity_stable iff continuity_error_t <= epsilon and T(M_eff) >= theta
Memento supplied this primitive: persistence alone is insufficient. A system may keep acting while
becoming manipulable if its continuity bridge is untrusted or corrupted.
4. Choice, Responsibility, Time
choice:
u_t = pi(x_t, o_t, M_eff, C_t, UQ_t)
responsibility_signal:
partial Omega_{t+1} / partial u_t != 0
time:
t < t+1
default := irreversible + path_dependent
Responsibility is separated into causal influence, foreseeability, constraint access, and correction access.
Causal influence may remain even when normative responsibility is degraded.
5. Correction Capacity
K = W * R * A * Tm * E * Rp
W witness / detection
R record / memory survival
A authority / access to alter course
Tm time before hardening
E enforcement / consequence carrier
Rp repair capacity
if any factor approx 0 -> K approx 0
This is an AND-gate, not a decorative equation. Do not turn K into a score unless each factor is
operationalised.

6. Clocks and Hardening
tau_action = time until action occurs
tau_detect = time until signal is noticed
tau_intervene = time until correction can affect the path
tau_harden = time until harm/dependency becomes practically irreversible
tau_method = time until method damages actor/institution/record
live_correction iff tau_intervene < tau_harden
late_review iff tau_intervene >= tau_harden
A safeguard is not judged only by existence. It must bite before hardening, and the safeguard itself must
remain corrigible.
7. Constraint and Meaning
large_state_space S_total
constraint C selects admissible region S_C subset S_total
meaningful_choice iff:
|S_C| > 1
|S_C| tractable
values/relations weight options
EEAAO supplied this primitive: overwhelming possibility without constraint collapses selection; meaning
appears inside bounded, weighted choice-space.
8. Future-Space and Hope
Xi_t := accessible_future_option_space
hope_signal := credible evidence that Xi_{t+k} is not empty
continuity_action := present cost paid to preserve Xi_future
false_hope := signal raises action but lacks carrier/route
Children of Men supplied this primitive: future-space can decay so far that present systems lose
coordination. A fragile future carrier can reorganise action if protection, timing, and trust hold.
9. Method as State-Changing Operator
m_t := method used to apply action u_t
x_{t+1}, Omega_{t+1} = f(x_t, Omega_t, u_t, m_t, xi_t)
method_valid iff:
m_t in C_method
and not body_as_lever(m_t)
and not innocent_relation_as_lever(m_t)
and record_integrity preserved
and correction_route remains live
Unthinkable supplies this primitive: method is not neutral plumbing. The way action is performed changes
evidence, subject state, actor state, institutional legitimacy, precedent, and future option-space.

10. Emergency and No-Clean-Branch
emergency_pressure := high DeltaH risk + short tau_harden + high UQ
valid_emergency_action requires:
credible threat_signal
no safer provisional state
method not categorically inadmissible
record preserved
independent review possible
repair route not destroyed
no_clean_branch := all available branches carry serious DeltaH
no_clean_branch != permission_to_use_any_method
TRACE can mark tragedy without laundering constraint deletion. Extreme stakes increase pressure; they
do not automatically convert inadmissible methods into admissible ones.
11. Uncertainty as First-Class
UQ_t := {unknowns, confidence, source_quality, scope, unresolved_category}
claim_status in {observation, hypothesis, model, measurement, finding}
every claim carries falsifier := what would break it
Uncertainty is part of state. It must not be erased by urgency, desire, story coherence, or compression
beauty.
```
