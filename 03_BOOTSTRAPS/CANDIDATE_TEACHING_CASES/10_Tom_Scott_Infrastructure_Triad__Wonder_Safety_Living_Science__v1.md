## Case A — Cragside / The First Smart Home

### Compression

```trace
Cragside_lesson :=
  the_future_can_arrive_locally
  before_the_correction_structure_arrives_globally
```

Cragside shows future luxury arriving early: hydraulic machinery, central heating, hydroelectricity, telephone links, incandescent lighting, and experimental domestic automation.

But the visible wonder is carried by hidden basements:

```trace
Cragside_basements :=
  servant_labour
  + child_risk
  + electrified_mercury
  + coal
  + imperial_arms_capital
  + distant_harm_pipeline
```

### TRACE Reading

```trace
technical_success != ethical_completion

smart_home :=
  convenience_for_owner
  + risk_for_staff
  + hidden_supply_chain
  + externalized_harm
```

The lamp system is the sharpest object:

```trace
lamp_system :=
  visible_output: light
  hidden_method:
    ten_year_old_handler
    + insulated_gloves
    + electrified_mercury
    + no_domestic_switch_standard
```

### K-Gate

```trace
K_Cragside_lamps :=
  Witness: partial
  Record: weak
  Authority: owner/engineer concentrated
  Time: immediate toxicity/electrocution risk
  Enforcement: immature/absent
  Repair: after_the_fact_only
```

### Main Supported Operators

- Method Floor
- Method Laundering
- Record Trust
- Legality versus legitimacy
- Basement routing
- Option-space diffusion

### Must Not Claim


## Case B — Network Rail Signalling / Why Trains Usually Do Not Crash

### Compression

```trace
rail_safety_law :=
  do_not_make_safety_depend_on_last_second_judgment
  when_you_can_make_dangerous_states_unavailable
```

This is the strongest direct TRACE-core case of the three.

Rail signalling does not rely on signallers merely being careful. It embeds safety into interlocks, block instruments, permission protocols, repeat-back communication, rear-lamp proof, abnormal-case training, and near-miss investigation.

### TRACE Reading

```trace
rail_signalling :=
  make_wrong_order_impossible
  + treat_unknown_as_danger
  + require repeated_permission
  + bind_communication_to_machinery
  + investigate_boundary_violations
  + train_for_abnormal_cases
```

### K-Gate

```trace
K_rail :=
  Witness: track_circuits + windows + diagrams + tail_lamps
  Record: signal_box_book + block_instruments + repeated_bell_codes
  Authority: signaller_can_clear_or_block_route
  Time: clearance_must_precede_block_entry
  Enforcement: mechanical/electrical_interlocks
  Repair: stop + investigate + reset + train + review_SPADs
```

### Core Safety Default

```trace
unknown != clear
unknown := danger
```

A track circuit does not primarily prove that a train is present. It proves, when clear, that a section can be treated as safe enough. When uncertain, the system blocks.

### Absolute Block

```trace
absolute_block :=
  one_train_per_signalled_block
  + no_following_train_without_protected_stop
```

Safety is not local emptiness. Safety is stoppable structure.

```trace
safe_route != empty_segment

safe_route :=
  empty_enough
  + signal_boundary
  + stopping_distance
  + authority_to_hold_next_train
```

### Main Supported Operators

- Correction Before Hardening
- K-Gate
- Harm Clock
- Method Floor
- Record Trust
- Safeguard Corrigibility
- Wrongability
- False-negative and near-miss discipline

### Must Not Claim

```trace
rail_case_must_not_claim :=
  old_systems_are_always_better
  OR modernisation_is_bad
  OR human_operators_are_enough
  OR fail_safe_design_eliminates_all_risk
```

## Case C — Jodrell Bank / Lovell Telescope

### Compression

```trace
living_infrastructure :=
  stable_core
  + replaceable_layers
  + repair_capacity
  + calibrated_record
  + future_question_capacity
```

Jodrell Bank shows that an old scientific instrument can remain live when its function is preserved through maintenance, upgrades, calibration, staff knowledge, and network integration.

### TRACE Reading

```trace
old != obsolete
heritage != dead
continuity != component_freeze
```

The Lovell Telescope remains useful because the core scientific form still works, while the surface, electronics, control systems, detectors, focal packages, and networked correlation have changed.

```trace
Lovell_Telescope :=
  old_core_structure
  + maintained_surface
  + upgraded_control
  + replaceable_detectors
  + networked_correlation
  + live_science
```

### K-Gate

```trace
K_Jodrell :=
  Witness: radio_signals + calibrators + pulsars + asteroids
  Record: raw_signals + correlated_data + timing_records + archives
  Authority: operators + engineers + observatory_governance
  Time: observing_windows + maintenance_cycles + infrastructure_lifespan
  Enforcement: control_procedures + radio_quiet_rules + access_controls
  Repair: painting + wheel_changes + surface_replacement + electronics_upgrades + detector_swaps
```

### Scientific Witness

```trace
scientific_witness :=

  + error_management
  + archive
```

### Heritage Tension

```trace
heritage_status :=

preservation_guard :=
```

### Main Supported Operators

## Triad Comparison


### Three Infrastructure Modes

| Case | Infrastructure mode | TRACE lesson | Risk |
|---|---|---|---|
| Cragside | early luxury infrastructure | wonder can hide basement routing | technical success cleans provenance |
| Rail signalling | mature safety infrastructure | safety should be embedded before hazard entry | nostalgia or interface conservatism becomes dogma |
| Jodrell Bank | living scientific infrastructure | repair and upgrade keep old systems alive | heritage freezes live function |
