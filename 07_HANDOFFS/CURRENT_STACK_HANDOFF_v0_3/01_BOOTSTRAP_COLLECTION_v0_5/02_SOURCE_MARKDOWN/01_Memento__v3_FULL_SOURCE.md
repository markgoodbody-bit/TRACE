1. Narrative Compression
Memento presents an agent who continues to act, reason, and pursue goals while lacking reliable
short-term memory. He externalises memory into notes, photographs, and tattoos. The film shows that
external memory can stabilise identity, but only if the memory bridge remains trustworthy. When the
