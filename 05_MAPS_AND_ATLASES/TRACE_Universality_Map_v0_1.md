Date: 2026-06-18
Status: scope architecture / applicability map / hostile-audit patched / not active spine / not validation / not proof / not operator promotion / not Kernel v0.3

## 0. Control header

This file gives TRACE a larger container without bloating the operator registry.

It does not add an operator.

It does not validate TRACE.

It does not create Kernel v0.3.

It does not prove TRACE is universal.

It does not promote any bootstrap case, film, story, or analogy into evidence.

It is a map for keeping the project wide while keeping the active spine narrow.

Hostile-audit patch, 2026-06-18: the word `universality` is now treated as an aspiration and test direction, not a property TRACE has earned.

## 1. Plain compression

TRACE is trying to become a general navigation grammar for systems under pressure.

It asks whether subjects, records, correction routes, and future options survive when power, time, infrastructure, and uncertainty interact.

The project can remain encompassing and ambitious. But scope does not mean one operator per domain.

```trace
scope_rule :=
  widen_domain
  + compress_spine
  + translate_locally
  + compare_hostilely
  + demote_relabels
```

Plain version:

The field stays wide. The spine stays small. Domains get translations. Failed local operators become useful notes rather than false primitives.

## 2. Love / Death / Robots as U0 mnemonic

The phrase is useful because it compresses three pressures TRACE keeps encountering.

It is not a new operator.

It is not evidence.

It is not a primitive-clustering engine.

It is a U0 recognition carrier: a front-door mnemonic for remembering the scope of the project.

```trace
LDR_status :=

  - evidence
```

## 3. TRACE scope stack

TRACE should separate primitives, operators, translations, cases, comparators, and formal bridges.

```trace
TRACE_scope_stack :=
  L0_primitives
  -> L1_active_operators
  -> L2_domain_translations
  -> L3_case_surfaces
  -> L4_comparators
  -> L5_formal_mechanistic_bridge
```

Do not collapse these layers.

```trace
Debt_Clock_after_hostile_audit :=
```

## 6. Domain translation registry role

A domain translation maps local language into the active spine without claiming a new operator.

```trace
domain_translation :=
  local_domain_language
  -> TRACE_primitives
  -> active_operator_if_applicable
  -> comparator
  -> demoter
```

The domain translation registry is a mapping layer and comparator-pressure surface. It is not an operator registry.

## 7. Applicability ladder

A TRACE idea should climb levels only by earning it.

```trace
applicability_ladder :=
  U0 := recognition_carrier
  U1 := domain_translation
  U2 := recurring_operator_candidate
  U3 := cross_domain_invariant
  U4 := formal_or_mechanistic_expression
  U5 := adversarially_comparable_universal_move
```

Level rules:

```trace
U0_story_or_case :=
  helps_reader_recognise_pattern
  - proof

U1_domain_translation :=
  maps_local_language_to_TRACE_spine
  - operator

U2_operator_candidate :=
  has_activation_condition
  + deactivation_condition
  + comparator
  + demoter
  + at_least_one_worked_delta_plan

U3_cross_domain_invariant :=
  survives_at_least_two_unrelated_domains
  + survives_prior_art_pressure
  + can_fail

U4_formal_or_mechanistic_expression :=
  measurable_or_modelled_variables
  + intervention_points
  + falsifiable_boundary

U5_universal_move :=
  improves_navigation_across_domains
  + beats_or_complements_existing_expert_reasoning
  + remains_demotable
```

No idea jumps from story to universal claim.

## 8. Formal / mechanistic bridge needed

TRACE still lacks a full formal and mechanistic-interpretability bridge.

The minimum bridge should define:

```trace
formal_bridge_needed :=
  state_vector
  + action_map
  + harm_delta
  + correction_window
  + hardening_function
  + agency_window
  + evidence_channel
  + enforcement_carrier
  + repair_function
```

Working sketch:

```trace
x_t := system_state_at_time_t

a_t := action_or_intervention_at_time_t

x_{t+1} := F(x_t, a_t, environment, power, constraint)

Delta_H := H(x_{t+1}) - H(x_t)

tau_correct := time_required_for_correction_to_reach_subject

tau_harden := time_until_loss_becomes_irreversible_or_materially_unrepairable

OP_001 := tau_correct < tau_harden
```

Mechanistic-interpretability bridge:

```trace
MI_bridge :=
  feature_or_circuit
  + activation_condition
  + causal_role
  + intervention_point
  + deployment_context
  + subject_harm_path
  + rollback_or_correction_channel
```

Plain version:

A model-internal explanation is not enough. TRACE asks whether interpretability changes the correction window before deployment hardens harm.

## 11. Tragic remainder layer needed

Some cases have no clean admissible branch.

TRACE can identify catastrophe, but it still needs a disciplined tragic-remainder layer for action under unavoidable harm.

```trace
tragic_remainder_needed :=
  no_admissible_branch
  + action_still_required
  + harm_minimisation
  + accountability
  + record_preservation
  + no_moral_victory_claim
  + repair_after_action
```

Plain version:

`no_admissible_branch` is not a decision. It is a failure-state marker. The operator still must act, but the action should not be laundered as clean.

## 16. Scope without bloat

The project can become more applicable precisely because it demotes local relabels.

```trace
scope_growth_rule :=
  primitives_expand_carefully
  + translations_expand_cautiously
  + comparators_expand
  + design_patterns_wait_until_delta
  while:
    active_operator_spine_remains_small
```

Plain version:

TRACE gets larger by mapping more domains onto fewer stronger moves.
