## Purpose

TRACE now has several atlas layers:

```trace
atlas_layers :=
  Positive_TRACE_Atlas
  + Life_Biology_Ecology_Atlas
  + Agency_Free_Will_Mandelbrot_Atlas
```

A cross-atlas pressure test asks whether those layers produce better structure when applied together.

```trace
pressure_test_question :=
  does_the_stack_add_structure
  OR merely_generate_elegant_language?
```
