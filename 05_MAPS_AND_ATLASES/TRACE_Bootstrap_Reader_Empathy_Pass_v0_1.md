# TRACE Bootstrap Reader Empathy Pass v0.1

Date: 2026-06-18
Status: support patch / reader-entry discipline / not installed in Bootstrap V2 / not validation / not operator promotion / not Kernel v0.3

## 0. Control header

This file gives targeted reader-entry insertions for the six Bootstrap V2 clusters.

It does not edit the 10-file Bootstrap V2 relay pack.

It does not add new story carriers.

It does not promote any story into evidence.

It does not validate TRACE.

It exists because Bootstrap V2 repeatedly warns against assuming the reader already knows the carrier story.

```trace
reader_empathy_pass :=
  carrier_identified
  + relevant_context_given
  + pattern_named_before_TRACE_translation
  + boundary_stated
  - fandom_dependency
  - proof_claim
```

## 1. Insertion rule

Each cluster should include a short reader-entry block near the top, after `Plain compression` and before `TRACE compression` or `Story carriers`.

The block should not summarise every case. It should give a non-fandom reader enough entry to understand why each story is present.

```trace
reader_entry_valid_if:
  reader_can_follow_pattern_without_prior_fandom_knowledge
  + story_context_is_minimal_but_sufficient
  + analogy_boundary_visible
```

## 2. Cluster 01 insertion — Memory, Identity, Recursion

Suggested insertion:

```markdown
## Reader-entry orientation

This cluster uses stories about memory and repetition as pressure diagrams, not as evidence. *Memento* is a film about a man whose short-term memory impairment forces him to rely on external notes, photographs, tattoos, and routines. *Groundhog Day* is a time-loop story in which repeated experience can train care or manipulation. *Everything Everywhere All at Once* uses branching realities to show how too many possible lives can collapse meaning unless relation and care restore direction.

The TRACE pattern is: memory, record, and repetition are live only when they improve correction. If they preserve confidence while corrupting the route back to truth, they become harm carriers.

Boundary: these stories do not prove psychological, legal, or AI-memory claims. They orient the reader before historical and technical source anchors do the evidential work.
```

## 3. Cluster 02 insertion — Hope, Future-Space, Collapse

Suggested insertion:

```markdown
## Reader-entry orientation

This cluster uses stories where continuation becomes uncertain. *Children of Men* imagines a world without children, where society continues materially but loses a carrier for future meaning. *Everything Everywhere All at Once* shows option-space becoming unbearable when every branch seems equally empty. Samwise from *The Lord of the Rings* carries practical hope by preserving the next step without seizing the burden as his own. Harriet Tubman and the Underground Railroad provide the serious historical pressure point: future-space can be a literal protected route out of predatory power.

The TRACE pattern is: hope is not optimism. Hope is usable future-space with a real carrier, route, capacity, relation, and repair path.

Boundary: story-carried hope must not become sentiment, reassurance, or a duty imposed on harmed subjects.
```

## 4. Cluster 03 insertion — Judgment, Uncertainty, Irreversible Harm

Suggested insertion:

```markdown
## Reader-entry orientation

This cluster uses stories where decisions may cause hard-to-reverse harm. *12 Angry Men* is a jury-room film in which one dissenting juror slows a rush toward conviction and forces the evidence to be re-tested. *Unthinkable* is an emergency-pressure story about whether extreme threat can launder forbidden methods. *Memento* appears here as a warning about decisive action from a corrupted record.

The TRACE pattern is: where harm is irreversible or very hard to repair, uncertainty cannot be treated as decoration. But the answer is not always delay; sometimes delay is itself the hardening path.

Boundary: this cluster does not say “always slow down.” It says tempo must be chosen according to which branch preserves correction before hardening.
```

## 5. Cluster 04 insertion — Power, Method, Coercion, Creator Responsibility

Suggested insertion:

```markdown
## Reader-entry orientation

This cluster uses stories where power tries to escape answerability. Data from *Star Trek: The Next Generation* is an artificial officer whose status is contested when others try to treat him as property or research material; he also recognises possible artificial subjects before certainty is settled. *Frankenstein* is a created-life story where the creator abandons responsibility after making a vulnerable being. *Unthinkable* tests whether emergency can suspend the method floor. Harriet Tubman provides the historical counter-pattern where legal authority itself is a harm carrier and protected exit becomes necessary.

The TRACE pattern is: power is not legitimate because it is effective. Power becomes safer only when method, subject position, correction, and exit remain live under pressure.

Boundary: Data and Frankenstein do not prove current AI personhood. They carry the pattern of classification laundering, created dependency, and responsibility under uncertainty.
```

## 6. Cluster 05 insertion — Energy, Infrastructure, Basement

Suggested insertion:

```markdown
## Reader-entry orientation

This cluster uses stories where survival depends on material and social carriers. Apollo 13 is the space-mission failure in which survival depends on telemetry, ground teams, procedures, power budgets, improvised repair, and time. *Greenland* shows disaster survival through ordinary infrastructure: medicine, transport, documents, gates, fuel, shelter, and bodies with dependencies. Harriet Tubman shows protected exit as route infrastructure under predatory law. Samwise shows care as human-scale support infrastructure that preserves agency rather than seizing it.

The TRACE pattern is: good outcomes require capacity, but capacity can hide costs in a basement. A system is not clean just because its visible surface works.

Boundary: this cluster is not anti-energy or anti-infrastructure. It distinguishes disclosed, contestable friction from hidden, non-contestable, downward-routed harm.
```

## 7. Cluster 06 insertion — Late Warning, Gated Survival

Suggested insertion:

```markdown
## Reader-entry orientation

This cluster uses disaster and escape stories to study warning, gates, selection, and timing. *2012* is a scientifically implausible but structurally useful spectacle about secret arks, elite access, and public warning arriving too late. *Greenland* is a more grounded family-scale disaster story where selection notices, medicine, military gates, separation, and route access decide survival. Harriet Tubman and the Underground Railroad provide the historical counter-pattern: when official gates are predatory, survival may require protected exit rather than appeal to the gate.

The TRACE pattern is: when time closes and survival capacity is gated, legitimacy depends on who knew, who was warned, who controlled the gate, who was left out, and whether errors could still be corrected.

Boundary: the films are pressure diagrams, not policy proof. This cluster must distinguish unavoidable scarcity from corrupt selection.
```
