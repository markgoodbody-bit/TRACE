## Why Maisie belongs here

The Positive TRACE Atlas has so far used story anchors:

```trace
Bluey -> Inside_Out -> Spirited_Away -> WALL_E -> The_Good_Place -> Paddington :=
  play
  -> emotion
  -> name
  -> world-repair
  -> moral-learning
  -> hospitality
```

Maisie moves the atlas from fiction into lived non-human care.

```trace
Maisie :=
  dog
  + family_member
  + ageing_subject
  + illness_subject
  + nonverbal_signal_carrier
  + daily_responsibility_anchor
```

Plain version: Maisie tests whether TRACE can map love and responsibility where the subject cannot argue, consent in human language, justify herself, or explain her experience. The care relation still binds.

## Care without contract

```trace
care_without_contract :=
  subject_cannot_bargain
  + subject_depends_on_carer
  + carer_controls_risk_path
  + duty_increases_not_decreases
```

Maisie does not owe usefulness to justify care. She is not a productivity unit, moral argument, or symbolic pet. She is a dependent subject in a relation.

```trace
nonhuman_care_rule :=
  dependency
  -> increased_responsibility
  not:
    decreased_value
```

This matters because many bad systems treat dependency as evidence of lower claim. TRACE should mark the opposite: when one side controls the harm-pathway, responsibility increases.

## Blindness / partial impairment

```trace
capacity_change :=
  sensory_loss_or_reduction
  + adaptation
  + environmental_support
  + unchanged_relation_claim
```

Partial blindness or lost sight in one eye does not erase Maisie. It changes how the environment must meet her.

```trace
dignity_under_impairment :=
  subject_still_same_subject
  + environment_adjusts
  + care_reads_new_limits
```

This is a strong positive TRACE primitive. Identity can persist through capacity change, but care must update.

## Temperament continuity

```trace
temperament_continuity :=
  recognisable_self
  + familiar_routines
  + relation_memory
  + ordinary_signals
```

The phrase “still herself” matters structurally. It is not proof of no suffering. It is a continuity signal: the subject still appears reachable, relational, and recognisable.

```trace
self_continuity_signal :=
  familiar_response_patterns
  + relational recognition
  + ordinary pleasure_or_comfort
```

Demoter:

```trace
Maisie_demoter_3 :=
  if_TRACE_uses_recognisable_self_to_ignore_pain
  then_mapping_failed
```

Continuity must not be used to deny suffering. It is a signal to interpret, not a permission to stop reading.

## End-of-life horizon

This case also contains the edge of death, even if not immediately present.

```trace
end_of_life_horizon :=
  love
  + dependency
  + pain_risk
  + dignity
  + timing
  + decision_burden
```

Nonhuman care forces a brutal responsibility: the carer may eventually have to choose not endless life, but least-bad mercy under uncertainty.

```trace
nonhuman_end_of_life_ethic :=
  avoid_unnecessary_suffering
  + preserve_comfort
  + seek_veterinary_truth
  + do_not_extend_for_owner_only
  + do_not_abandon_too_soon
```

This should remain under humility. TRACE must not pretend to solve veterinary or end-of-life judgement.

```trace
end_of_life_demoter :=
  if_TRACE_confuses_owner_grief_with_subject_welfare
  OR treats_death_as_always_failure
  then_mapping_failed
```
