## 7. Cross-domain abstract form that survived

The current common form is not a single equation of everything. It is a typed translation discipline:

```trace
for_each_domain_D:
  choose native_case x_D
  identify native_structure S_D,T_D,C_D,O_D,K_D,B_D,I_D
  build profile P_D(x_D)
  translate via candidate map m_D
  attach loss L_D
  test BT_D(m_D(x_D))
  demote if loss hidden or back_translation_fails
```

This is the strongest current unification statement.
