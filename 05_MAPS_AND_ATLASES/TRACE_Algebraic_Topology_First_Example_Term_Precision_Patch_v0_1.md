## 3. Boundary term disambiguation

The word `boundary` has three different roles in this branch.

```trace
boundary_terms :=
  topological_boundary_of_space
  + chain_boundary_under_boundary_operator
  + TRACE_boundary_role_B_Top
```

### 3.1 Topological boundary of a space

```trace
topological_boundary_of_space :=
  boundary_as_part_of_space_or_subspace_description
  example:
    disk_D2_has_boundary_circle
```

Plain version:

This refers to the edge-like boundary of a space such as the circle bounding a disk.

### 3.2 Chain boundary under a boundary operator

```trace
chain_boundary_under_boundary_operator :=
  boundary_as_image_of_higher_dimensional_chain_under_boundary_operator
  example:
    a_loop_may_bound_a_filled_2_chain_in_a_space
```

Plain version:

This is the homology-relevant sense. It asks whether a cycle is the boundary of a higher-dimensional chain.

### 3.3 TRACE boundary role B_Top

```trace
TRACE_boundary_role_B_Top :=
  translation_role_for_scope_boundary_context_or_boundary_operator
  not:
    identical_to_topological_boundary
    OR identical_to_chain_boundary
```

Plain version:

TRACE `B_Top` is a role slot. It must point to the relevant native boundary structure rather than replacing native terminology.
