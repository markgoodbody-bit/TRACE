# TRACE / ME Concordance v0.6

Status: mapping layer. Not validation. Not proof.

## Purpose

The Concordance prevents TRACE from laundering novelty. Each operator must be compared with existing work and must state any distinct remainder.

```trace
concordance :=
  TRACE_operator
  + established_counterpart
  + what_TRACE_adds_if_any
  + demoter_if_relabel_only
```
