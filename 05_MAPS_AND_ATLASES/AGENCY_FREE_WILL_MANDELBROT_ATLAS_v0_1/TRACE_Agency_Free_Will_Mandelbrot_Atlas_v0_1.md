## Selection under constraint

```trace
selection :=
  option_space
  + weight_vector
  + constraint_set
  + pressure_field
  -> chosen_path
```

Selection is never from infinite freedom. It occurs inside a shaped option-space.

```trace
option_space :=
  available_paths
  - impossible_paths
  - blocked_paths
  - unseen_paths
```

TRACE should always ask who shaped the option-space.

```trace
chooser_legitimacy_question :=

  + who_controls_weights?
  + who controls constraints?
  + who pays consequence?
```

## Responsibility after selection

Responsibility attaches most clearly where effective control meets foreseeable consequence.

```trace
responsibility :=
  effective_control
  + foreseeable_effect
  + selection_or_maintenance_of_pathway
  + capacity_to_reduce_harm
```

This avoids two errors:

```trace
responsibility_errors :=
  blame_without_control
  OR no_responsibility_because_system_complex
```

Responsibility does not require total control. It requires enough control over a harm-pathway to bind future action.

```trace
partial_control_responsibility :=
  control_over_some_lever
  + knowledge_or_reason_to_know
  + ability_to_reduce_risk
```

## Degrees of agency

Agency is graded.

```trace
agency_gradient :=
  reactive
  -> adaptive
  -> selective
  -> reflective
  -> self-correcting
```

### Reactive

```trace
reactive :=
  stimulus
  -> response
```

### Adaptive

```trace
adaptive :=
  response_changes_with_feedback
```

### Selective

```trace
selective :=
  options_modeled
  + path_chosen
```

### Reflective

```trace
reflective :=
  self_model
  + reason_about_selection
```

### Self-correcting

```trace
self_correcting :=
  recognise_error
  + update_constraints_or_weights
  + reduce_repetition
```

Do not collapse these into one category.

## Freedom and constraint

Freedom is not absence of all constraint. A system with no constraints cannot sustain meaningful action.

```trace
freedom :=
  meaningful_option_space
  + usable_information
  + non-coerced_selection
  + viable_exit_or_contest
```

False freedom:

```trace
false_freedom :=
  many_options
  + hidden coercion
  + no viable repair path
  + burden shifted onto weaker subject
```

This links to policy, markets, platforms, labour, and family systems.

```trace
coercion :=
```
