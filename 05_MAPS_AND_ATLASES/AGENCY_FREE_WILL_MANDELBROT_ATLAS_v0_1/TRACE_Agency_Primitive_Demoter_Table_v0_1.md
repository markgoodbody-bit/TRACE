# TRACE Agency Primitive / Demoter Table v0.1

Status: compact companion to `TRACE_Agency_Free_Will_Mandelbrot_Atlas_v0_1.md`. Working table only. Not canon. Not validation. Not a proof of free will. Not operator promotion. Use as a compression surface for mapping agency as constrained selection with consequence and possible correction.

## Purpose

The Agency / Free Will / Mandelbrot Atlas seeded this frame:

```trace
agency :=
  options
  + weights
  + constraints
  + pressure
  + memory
  + selection
  + consequence
  + update
```

This table keeps that frame usable and prevents three common errors:

```trace
risk_set :=
  metaphysical_overclaim
  + blame_without_control
  + predation_or_coercion_overuse
```

Plain version: TRACE can map choice without pretending to solve free will. It can assign responsibility only where control, consequence, and possible correction have been inspected.

## Compact table

| Primitive | Function | Main failure / demoter |
|---|---|---|
| `Aperture_Gate` | Locates where modelled inputs narrow into selected causal action. | Fails if treated as consciousness, personhood, soul, or moral-floor proof. |
| `selection_under_constraint` | Maps choice as option-space + pressure + constraint + selected path. | Fails if choice is treated as unconstrained magic or as no-choice mechanism. |
| `option_space` | Shows what paths were available, blocked, unseen, or made impossible. | Fails if responsibility is assessed without asking who shaped the options. |
| `weight_vector` | Captures priorities, incentives, fears, loyalties, habits, and salience. | Fails if all options are treated as equally live or equally weighted. |
| `pressure_field` | Tracks time, scarcity, threat, fatigue, incentive, and social force. | Fails if selection is analysed as if made under calm conditions. |
| `constraint_quality` | Separates good constraints that preserve future-space from bad constraints that trap subjects. | Fails if all constraint is treated as oppression or all constraint as safety. |
| `memory_training_loop` | Links past selection, feedback, learning, and future change. | Fails if selection leaves no trace or update possibility. |
| `consequence_binding` | Binds future action to effects caused or maintained by selection. | Fails if recognition produces no changed future action. |
| `responsibility_after_selection` | Assigns responsibility where control, foreseeability, and harm-reduction capacity exist. | Fails as blame without control or no responsibility because system is complex. |
| `Mirror_Gate` | Locates self-recognition of selection and effect on others. | Fails if self-recognition remains theatre and does not bind future conduct. |
| `recursive_selection_pattern` | Finds the same selection structure across person, family, institution, state, ecology, and AI system. | Fails if Mandelbrot similarity is rhetorical and does no analytical work. |
| `agency_gradient` | Distinguishes reactive, adaptive, selective, reflective, and self-correcting agency. | Fails if all agency is collapsed into one category. |
| `nonverbal_agency` | Reads agency through preference, avoidance, seeking, learning, routine, and relation signals. | Fails if language absence is treated as agency absence, or projection replaces signal-reading. |
| `AI_selection_map` | Maps AI output/tool selection without assuming inner subjecthood. | Fails if selection map becomes personhood claim or if agency-like effects are ignored entirely. |
| `meaningful_freedom` | Defines freedom as usable option-space, information, non-coerced selection, and viable exit/contest. | Fails if many options hide coercion or if freedom is defined as no constraint. |
| `coercion` | Tracks artificial narrowing of options through threat, dependency, or burden routing. | Fails if all influence is called coercion or if coercion is missed because a choice surface exists. |
| `chooser_legitimacy` | Asks who has the right, standing, or role to control selection. | Fails if selection authority is assumed from power alone. |
| `option_shaping` | Tracks upstream design of what choices a subject can see or take. | Fails if only final choice is assessed while prior shaping is ignored. |
| `update_or_defence` | Distinguishes learning from rationalisation after consequence. | Fails if a system defends the old pathway while claiming to have learned. |

## Free will guard

```trace
free_will_guard :=
  metaphysical_question_open
  + practical_selection_real_enough_for_accountability
  + no_total_blame_without_control
  + no_total_escape_because_causality
```

Plain version: TRACE should not claim that a person, institution, or AI is metaphysically free in the ultimate sense. It should ask what selection occurred, who shaped it, what consequence followed, and what correction is now possible.

## Responsibility test

```trace
responsibility_test :=
  what_control_existed?
  + what_effect_was_foreseeable?
  + what_pathway_was_selected_or_maintained?
  + what_harm_reduction_was_possible?
  + what_update_is_now_required?
```

If these are absent, responsibility weakens. If they are present, complexity alone does not erase responsibility.

## Mandelbrot use test

```trace
Mandelbrot_use_test :=
  does_pattern_recur_across_scales?
  + does_recurrence_clarify_selection?
  + does_it_clarify_pressure?
  + does_it_clarify_consequence?
  + does_it_clarify_correction_or_defence?
```

If not, do not use Mandelbrot language.
