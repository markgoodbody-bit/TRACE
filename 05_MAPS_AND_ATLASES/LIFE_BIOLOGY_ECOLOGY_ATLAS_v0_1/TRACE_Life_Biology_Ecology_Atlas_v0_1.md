## Why this file exists

The Positive TRACE Atlas began to map what TRACE is protecting: play, home repair, mixed emotion, name preservation, embodied attention, moral learning, hospitality, nonhuman care, and fragile future-space.

The next missing layer is life itself.

```trace
missing_layer :=
  life
  + biology
  + ecology
  + embodied_constraint
```

TRACE has often mapped institutions, timing, correction channels, hidden bills, AI systems, and human meaning. But those are downstream of biological reality. Living systems are the original carriers of boundary, energy, repair, memory, selection, dependency, predation, ageing, and death.

```trace
biology_importance :=
  ethics_does_not_float_above_life
  + correction_depends_on_living_process
  + harm_is_often_future_space_collapse_in_living_systems
```

```trace
living_entity :=
```

## Boundary

```trace
biological_boundary :=
  inside_outside_distinction
  + selective_exchange
  + vulnerability_surface
  + identity_condition
```

Every living system has some way of distinguishing self from environment, even if porous, nested, or unstable.

```trace
boundary_not_wall :=
  protects
  + filters
  + exchanges
  + adapts
```

This matters for ethics because harm often crosses boundaries: skin, immune system, home, consent, ecosystem, informational boundary.

```trace
boundary_violation :=
  forced_exchange
  OR blocked_needed_exchange
  OR boundary_destroyed
```

## Homeostasis

```trace
homeostasis :=
  regulated_range
  + feedback
  + correction
  + tolerance_band
```

Homeostasis is not stasis. It is active correction around a viable range.

```trace
healthy_stability :=
  fluctuation
  + feedback
  + correction_before_threshold
```

This gives TRACE a biological analogue for institutional correction:

```trace
correction_before_hardening_biological :=
  detect_deviation
  + respond_early
  + avoid_systemic_cascade
```

Failure mode:

```trace
homeostatic_failure :=
  signal_missed
  OR correction_too_late
  OR feedback_loop_overreacts
```

## Pain and signal

```trace
pain_signal :=
  damage_or_threat_information
  + action_pressure
  + protection_trigger
```

Pain is not identical with harm. Pain can prevent harm by forcing attention. But pain can become harm when it is chronic, unrepairable, ignored, or used as control.

```trace
healthy_pain :=
  signal
  + route_to_repair

harmful_pain :=
  signal_trapped
  + no_repair_path
  + forced_exposure
```

This links back to Inside Out and Maisie.

```trace
immune_failure_modes :=
```

## Parasitism

```trace
parasitism :=
  host_dependency
  + extraction
  + concealed_or_managed_damage
  + continued_host_viability_needed
```

Parasitism differs from immediate predation because the parasite often needs the host alive.

```trace
parasitic_system :=
  extract_without_killing_fast
  + keep_host_functional_enough
  + route_damage_to_host
```

This maps many institutional and economic patterns, but must be used carefully.

## Carrying capacity and overshoot

```trace
carrying_capacity :=
  system_limit

  + population_or_load
  + waste_absorption
```

Overshoot is delayed harm.

```trace
overshoot :=
  consumption_rate
```

## Death

```trace
death :=
  organism_continuity_end
  + relation_reconfiguration
  + memory_transfer_or_loss
  + future_path_closure
```

Death is not always the same as harm, but killing, preventable death, abandoned dying, and erased witness are ethically heavy.

```trace
death_distinctions :=
  natural_end
  + preventable_loss
  + mercy_release
  + violent_foreclosure
  + extinction
```

TRACE must avoid two errors:

```trace
death_errors :=
  treat_all_death_as_same
  OR treat_death_as_only_biological_event
```

Death is biological, relational, temporal, and archival.
