## Compact table

| Primitive | Biological pressure | TRACE function | Main failure / demoter |
|---|---|---|---|
| `active_persistence` | Living things must keep doing work to remain themselves. | Prevents treating life as static object. | Fails if TRACE treats a living system as inert structure rather than maintained process. |
| `biological_boundary` | Life requires inside/outside distinction and selective exchange. | Maps vulnerability, filtering, exchange, and identity. | Fails if boundary is treated as wall-only or ignored when forced exchange occurs. |
| `metabolism` | Life captures energy, transforms it, and routes waste. | Grounds harm/repair in energy and material cost. | Fails if TRACE ignores energy, waste, maintenance cost, or material limits. |
| `homeostasis` | Viability depends on regulated ranges and feedback. | Gives biological model for correction before cascade. | Fails if stability is mistaken for stillness, or feedback over/under-response is missed. |
| `pain_signal` | Pain can warn of damage or threat. | Separates signal from harm while preserving suffering. | Fails if all pain is treated as harm, or pain is ignored because it is “only signal.” |
| `immune_response` | Defence requires threat detection, memory, and proportional response. | Maps protection that can also harm. | Fails if protective response is assumed good without self-damage/proportionality check. |
| `biological_repair` | Damage requires resource allocation and restoration/scar. | Separates repair from erasure/reset. | Fails if survival is treated as full restoration or scar as mere failure. |
| `formation_window` | Developing systems have timing-sensitive phases. | Names why early harm can reshape future-space. | Fails if all harm timing is treated as equal. |
| `reproduction_or_continuity_pressure` | Life often generates future subjects or continuity routes. | Connects life to future-space without reducing value to reproduction. | Fails if non-reproductive life is treated as lower value. |
| `symbiosis` | Different life forms can share survival paths. | Maps positive dependency and mutual stabilisation. | Fails if all dependency is read as capture or all mutualism is assumed stable. |
| `predation_pattern` | One life uses another as resource. | Names fast extraction and consumed future-space. | Fails if natural predation is turned into moral permission, or all asymmetry is called predation. |
| `parasitic_system` | Extraction may require keeping the host alive. | Maps concealed burden and managed damage. | Fails if slow extraction is missed because host still functions. |
| `ecological_hidden_bill` | Waste/damage can be deferred into the system. | Connects ecology to hidden-bill and debt-clock logic. | Fails if local extraction is analysed without downstream system return. |
| `carrying_capacity` | Systems have regeneration and absorption limits. | Maps finite load before collapse. | Fails if limits are treated as optional preferences. |
| `biological_debt_clock` | Damage accumulates while repair/regeneration lags. | Gives biological form of harm-clock / repair-clock mismatch. | Fails if delayed collapse is treated as sudden or uncaused. |
| `ageing_dignity` | Repair capacity and vulnerability change over time. | Preserves subject value through capacity change. | Fails if ageing or impairment is treated as value loss. |
| `death_distinctions` | Death can be natural end, preventable loss, mercy release, violent foreclosure, or extinction. | Prevents crude “death = one category” mapping. | Fails if all death is treated the same, or death is reduced to biology only. |

## Primitive notes

### 1. Active persistence

```trace
active_persistence :=
  persistence_requires_work
```

Living continuity is not passive presence. It requires ongoing energy, maintenance, exchange, and repair.

Demoter:

```trace
active_persistence_demoter :=
  if_life_treated_as_static_object
  then_mapping_failed
```

### 2. Biological boundary

```trace
biological_boundary :=
  inside_outside_distinction
  + selective_exchange
  + vulnerability_surface
  + identity_condition
```

Boundary is not isolation. A living boundary filters exchange.

Demoter:

```trace
boundary_demoter :=
  if_boundary_treated_as_wall_only
  OR_forced_exchange_ignored
  then_mapping_failed
```

### 3. Metabolism

```trace
metabolism :=
  energy_capture
  + transformation
  + waste_routing
  + maintenance
```

No energy, no repair. No waste routing, hidden bill.

Demoter:

```trace
metabolism_demoter :=
  if_energy_and_waste_disappear
  then_TRACE_becomes_vibes
```

### 4. Homeostasis

```trace
homeostasis :=
  regulated_range
  + feedback
  + correction
  + tolerance_band
```

Healthy stability is active correction, not stillness.

Demoter:

```trace
homeostasis_demoter :=
  if_stability_mistaken_for_no_change
  OR_feedback_overreaction_ignored
  then_mapping_failed
```

### 5. Pain signal

```trace
pain_signal :=
  damage_or_threat_information
  + action_pressure
  + protection_trigger
```

Pain may protect life by forcing attention. Pain becomes harm when trapped, chronic, ignored, or used as control.

Demoter:

```trace
pain_demoter :=
  if_pain_equals_harm_always
  OR_pain_dismissed_as_only_signal
  then_mapping_failed
```

### 6. Immune response

```trace
immune_response :=
  boundary_defence
  + threat_detection
  + memory
  + proportional_response
```

Protection can damage the subject if misdirected or excessive.

Demoter:

```trace
immune_demoter :=
  if_defence_assumed_good_without_self_damage_check
  then_mapping_failed
```

### 7. Biological repair

```trace
biological_repair :=
  damage_detection
  + resource_allocation
  + restoration_where_possible
  + scar_where_needed
```

Repair is not always reset. Scar can be stabilised continuity.

Demoter:

```trace
repair_demoter :=
  if_repair_treated_as_damage_erasure
  then_mapping_failed
```

### 8. Formation window

```trace
formation_window :=
  timing_sensitive_growth_phase
  + high_future_weight
```

Harms during formation can reshape future option-space more deeply.

Demoter:

```trace
formation_demoter :=
  if_timing_of_harm_treated_as_irrelevant
  then_mapping_failed
```

### 9. Reproduction or continuity pressure

```trace
reproduction_or_continuity_pressure :=
  lineage
  + care
  + teaching
  + memory
  + ecological_relation
  + cultural_transfer
```

Continuity is not only genetic reproduction.

Demoter:

```trace
continuity_demoter :=
  if_life_value_reduced_to_reproduction
  then_mapping_failed
```

### 10. Symbiosis

```trace
symbiosis :=
  different_life_forms
  + mutual_or_asymmetric_dependency
  + shared_survival_path
```

Some dependency increases future-space rather than reducing it.

Demoter:

```trace
symbiosis_demoter :=
  if_all_dependency_read_as_capture
  OR_mutualism_assumed_capture_free
  then_mapping_failed
```

### 11. Predation pattern

```trace
predation_pattern :=
  fast_extraction
  + weak_target_correction
  + consumed_future_space
```

Predation is natural in ecology but not automatically morally transferable.

Demoter:

```trace
predation_demoter :=
  if_natural_predation_used_as_moral_permission
  OR_all_asymmetry_called_predation
  then_mapping_failed
```

### 12. Parasitic system

```trace
parasitic_system :=
  host_dependency
  + extraction
  + concealed_or_managed_damage
  + continued_host_viability_needed
```

The host functioning does not prove the relation is healthy.

Demoter:

```trace
parasite_demoter :=
  if_host_still_functioning_means_no_harm
  then_mapping_failed
```

### 13. Ecological hidden bill

```trace
ecological_hidden_bill :=
  extract_now
  + defer_waste_or_damage
  + future_system_pays
```

Ecology forces delayed-return thinking.

Demoter:

```trace
ecology_demoter :=
  if_downstream_damage_excluded_from_accounting
  then_mapping_failed
```

### 14. Carrying capacity

```trace
carrying_capacity :=
  system_limit
  + resource_regeneration_rate
  + population_or_load
  + waste_absorption
```

Carrying capacity is a constraint, not a mood.

Demoter:

```trace
capacity_demoter :=
  if_limits_treated_as_preferences
  then_mapping_failed
```

### 15. Biological debt clock

```trace
biological_debt_clock :=
  damage_accumulates
  while:
    repair_or_regeneration_lags
```

Delayed collapse can be slow injury, not sudden mystery.

Demoter:

```trace
debt_clock_demoter :=
  if_delayed_collapse_treated_as_sudden_unrelated_event
  then_mapping_failed
```

### 16. Ageing dignity

```trace
ageing_dignity :=
  capacity_changes
  + subject_value_remains
  + environment_and_care_update
```

Capacity change requires support update, not value demotion.

Demoter:

```trace
ageing_demoter :=
  if_capacity_loss_treated_as_value_loss
  then_mapping_failed
```

### 17. Death distinctions

```trace
death_distinctions :=
  natural_end
  + preventable_loss
  + mercy_release
  + violent_foreclosure
  + extinction
```

Death is biological, relational, temporal, and archival.

Demoter:

```trace
death_demoter :=

  then_mapping_failed
```

## Metaphor guard

```trace
metaphor_guard :=
  biology_can_inform_structure
  but:
    metaphor != evidence
    analogy != proof
    natural != moral
```

Plain version: life gives TRACE powerful structure, but biology must not become decorative authority. A biological analogy can suggest a pattern. It does not prove an ethical rule by itself.
