## Source phrase / carrier

A Tennessee Williams quotation supplied in conversation frames the world as violent and mercurial, and says that what must be saved from the perpetually burning building is love: love for each other and love poured into art, parenting, writing, painting, and friendship.

This file does not use the quotation as evidence. It uses it as a compression carrier.

## Candidate operator — Preservation Love

```trace
preservation_love :=
  costly_care_action
  that_preserves:
    life
    + voice
    + memory
    + relation
    + future_space
    + beauty
    + shared_world
  under:
    loss_pressure

    + violence
    + time_pressure
```

### Definition

Preservation Love exists where an agent or subject pays a real cost to preserve another subject, a shared world, a record, a possibility, or a continuity route under conditions where abandonment or self-protection would be easier.

It is not an emotion claim. It is a pressure-tested action pattern.

```trace
love_claim_becomes_TRACE_relevant_if:
  love_changes_what_is_protected
  under_pressure
```

## Relation to free will and responsibility

Preservation Love is connected to agency because it often keeps a route open for someone whose agency is threatened.

```trace
agency_effect :=
  preserves_or_reopens:
    option_space
    + voice
    + memory
    + contestability
    + relation
```

It connects to responsibility because it often appears where someone recognises they can carry a burden without owning the whole world.

```trace
responsibility_effect :=
  accepts_limited_cost
  to_prevent:
    avoidable_route_collapse
```

But it must not become infinite duty.

```trace
limit_rule :=
  care_subject_includes_carer
```
