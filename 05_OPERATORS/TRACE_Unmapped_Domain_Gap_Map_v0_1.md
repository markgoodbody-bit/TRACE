# TRACE Unmapped Domain Gap Map v0.1

Date: 2026-06-17
Status: gap map / operator-probe routing / not validation / not case expansion

## Plain purpose

This file identifies major domains TRACE has not yet mapped cleanly.

It does not add new operators. It routes future probes, names likely carrier stories, and prevents case sprawl by requiring every future story to justify itself through an operator, comparator, demoter, or test.

```trace
gap_map :=
  find_under_mapped_domains
  + list_candidate_story_carriers
  + define_operator_questions
  + hold_promotion_thresholds
  - case_sprawl
  - proof_claim
  - book_drafting
```

## Current mapped strength

TRACE is currently strongest around:

```trace
mapped_strengths :=
  gate
  + clock
  + live_record
  + correction_before_hardening
  + lever_realism
  + responsibility_laundering
  + affected_subject_route
  + actor_controlled_evidence_warning
```

It is weakest where harm is not primarily institutional, technical, or bureaucratic.

## Promotion rule

No gap becomes a new TRACE branch unless it survives this test:

```trace
promote_gap_only_if:
  candidate_operator_exists
  + comparator_exists
  + non_story_case_possible
  + demoter_named
  + adds_more_than_plain_language
```

Story carriers are recognition surfaces, not evidence.

```trace
story_rule :=
  story_can_reveal_pattern
  but:
    story_not_proof
    + story_not_validation
    + story_not_historical_substitute
```
