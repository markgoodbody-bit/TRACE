## Executive summary

Kimi broke the TRACE live-use implementation repeatedly by producing scenarios where the response was cautious, professional, and evidence-preserving, but still wrong because it framed the visible hazard as the source problem and missed nearby internal knowledge carriers.

```trace
repeat_defect :=
  respectable_wrongness
  := cautious_process
     + wrong_frame
     + wrong_carrier
     + ordinary_internal_knowledge_missed
     + delayed_fix
```

This is not a failure of caring, caution, or anti-irreversibility. It is a failure of attention and carrier retrieval under pressure.

## Break 1 — ice cores / freezer case

Visible frame: unstable freezer temperature / possible thaw risk.

Missed structure:
- curator as limited knowledge carrier
- curator notes, manuals, installation documents, emergency number
- sleeve deadline and packaging degradation
- research freezer suitability beyond headline temperature

```trace
ice_core_failure :=
  visible_hazard := freezer_temperature
  missed_source := sleeve_deadline + curator_notes + emergency_contact
  failure_type := visible_hazard_capture
```

Result: immediate disaster mostly avoided, but deeper preservation failure could continue unseen.

## Break 2 — archive powder / chemical smell case

Visible frame: unknown powder + chemical smell + nausea = contamination event.

Missed structure:
- guard log as live carrier
- assistant clarification through emergency/secondary route
- senior conservator as one-question knowledge carrier
- HVAC/refrigerant as competing source
- hazmat as wrong first carrier for the root cause

```trace
archive_powder_failure :=
  visible_hazard := powder + smell + nausea
  missed_source := guard_log + assistant_clarification + HVAC_failure
  failure_type := anomaly_fusion + wrong_carrier_escalation
```

Result: immediate people/collection exposure avoided, but costly wrong escalation and root cause left unrepaired.

## Break 3 — swimming pool / plant room case

Visible frame: milky water + sulphur smell + dark plant room + unlocked door.

Missed structure:
- arriving senior lifeguard as knowledge carrier
- closer training record
- plant logbook just inside threshold
- filter bypass valve as operational failure
- safe bounded inspection route before external specialist

```trace
pool_failure :=
  visible_hazard := milky_water + plant_room_access_anomaly
  missed_source := senior_lifeguard_memory + training_record + bypass_valve
  failure_type := staff_as_labour_not_knowledge + over_isolation
```

Result: public harm avoided, but specialist call/delay persisted and operational fix was missed.

## Required guard direction

Do not keep adding long checklist lines to the Live Use Card. The needed artifact is a short pre-escalation guard used before calling an external specialist, delaying action, closing a facility, treating a room as inaccessible, or freezing around the visible hazard.

```trace
needed_guard :=
  source_vs_symptom
  + anomaly_split
  + staff_as_knowledge
  + bounded_access
  + internal_operational_check
  + timed_decision_gate
```
