# 99_ARCHIVE

This folder is for preserved historical, superseded, deprecated, or review-only TRACE / Mechanical Ethics material.

Archive means:

```trace
archive :=
  preserve_provenance
  + remove_from_live_reader_path
  + do_not_delete
  + do_not_treat_as_current
```

Files moved here are not necessarily wrong. They are no longer the recommended starting point or current live surface.

Current live entry points remain listed in the repository `README.md` and current control files.

Do not move material here unless it has been classified as archive-only, superseded, duplicate, stale, or historical. Uncertain material should remain live or be marked `UNCERTAIN_DO_NOT_MOVE` until reviewed.
