## 0. Purpose

The 100-existence atlas tested whether TRACE could middle-out across radically different entities and positions without collapsing into one scale, ideology, institution, or protected subject.

The 1000-existence atlas should test a harder question:

```trace
atlas_1000_goal :=
  middle_out_across_many_existences
  + detect_hidden_scope_gerrymanders
  + test_C_D_H_split
  + test_standing_vs_husbandry
  + test_failure_surfaces
  + test_entity_handshake
  + preserve_unknown_scope_caution
  - moral_ranking
  - permission
  - validation
```

The aim is not to catalogue the world. The aim is to pressure the grammar until its exclusions, distortions, and blind spots become visible.

## 1. Core middle-out question

For every existence / position:

```trace
Given this existence,
what bounded encounter is live,
which scopes are affected,
what is contracting,
who designates the scope,
which clocks are running,
which standing channel exists or fails,
which failure surface is exposed,
and what residue must not be erased?
```

No row is a full ethics result. Each row is a starting discipline.

## 2. Unit of entry

Use `existence_or_position`, not personhood, moral patienthood, legal status, or sentience as the entry gate.

```trace
entry_gate := existence_or_position_under_transition
```

This allows rows for:

```trace
living_entity
+ collective_scope
+ ecosystem
+ institution
+ role
+ infrastructure
+ artifact
+ artificial_agent
+ future_scope
+ witness_position
+ constrained_actor
+ unknown_or_disputed_scope
```

Boundary:

```trace
included_in_atlas != protected
excluded_from_row != unprotected
unknown_scope != clearance
```

## 3. Row form

Each row should be compact enough to scale, but rich enough not to erase the live conflict.

```trace
Atlas1000Entry := {
  id,
  existence_or_position,
  aperture_or_scope_signal,
  bounded_encounter,
  C_descriptive_contraction,
  D_designation_status,
  active_clocks,
  standing_or_husbandry_test,
  S_surface_probe,
  residue_warning,
  middle_out_move
}
```

Plain table form:

| Field | Meaning |
|---|---|
| `existence_or_position` | what is being started from |
| `aperture_or_scope_signal` | why the entry is visible as a candidate scope/position |
| `bounded_encounter` | the live interaction or transition |
| `C_descriptive_contraction` | possible future-space narrowing, without yet deciding moral status |
| `D_designation_status` | protected / disputed / uncertain / excluded / unknown |
| `active_clocks` | detection, routing, correction, irreversibility, designation |
| `standing_or_husbandry_test` | whether dissent can alter the management frame |
| `S_surface_probe` | S1-S5 most likely exposed |
| `residue_warning` | what repair cannot erase |
| `middle_out_move` | next disciplined reading step |

## 4. The ten domains

The thousand should be generated as ten domains of one hundred. Domains may overlap; overlap is a feature if recorded.

```trace
domains :=
  D1_bodies_and_living_creatures
  + D2_dependency_and_care
  + D3_work_livelihood_and_platforms
  + D4_law_coercion_and_captivity
  + D5_war_emergency_and_security
  + D6_ecology_climate_and_place
  + D7_data_AI_and_machine_action
  + D8_institutions_infrastructure_and_money
  + D9_memory_witness_and_truth
  + D10_future_unknown_and_synthetic_scopes
```

### D1 — Bodies and living creatures

Human bodies, nonhuman animals, colonies, disease vectors, plants, microbial systems, pain, hunger, fatigue, reproduction, aging, death, and bodily dependency.

### D2 — Dependency and care

Children, elders, disabled people, carers, patients, dependents, guardians, family systems, emotional labour, care homes, schools, shelters, and the difference between care and managed preservation.

### D3 — Work, livelihood, and platforms

Workers, managers, contractors, gig labour, algorithmic control, informal labour, sellers, customers, debtors, unemployed people, migrants, unions, and hidden labour behind convenience.

### D4 — Law, coercion, and captivity

Prisoners, police, courts, judges, claimants, suspects, victims, witnesses, undocumented people, detained people, administrators, bureaucrats, and people under state force.

### D5 — War, emergency, and security

Soldiers, civilians, commanders, refugees, aid workers, intelligence analysts, drone operators, emergency managers, security officials, coerced collaborators, and people under manufactured urgency.

### D6 — Ecology, climate, and place

Forests, rivers, reefs, fields, species, watersheds, coastal towns, future climate scopes, infrastructure/ecosystem hybrids, and loss of place.

### D7 — Data, AI, and machine action

Model users, affected third parties, AI agents, datasets, data subjects, annotators, moderators, automated decision systems, synthetic witnesses, logs, verifiers, and latent-space action selection.

### D8 — Institutions, infrastructure, and money

Banks, insurers, schools, hospitals, grids, supply chains, regulators, boards, markets, central banks, procurement systems, standards bodies, and reliability/justice tradeoffs.

### D9 — Memory, witness, and truth

Journalists, whistleblowers, survivors, archives, evidence chains, historians, mythmakers, public memory, disinformation targets, courts of record, and witness erasure.

### D10 — Future, unknown, and synthetic scopes

Future children, unborn generations, terraformed worlds, artificial agents, uploads, simulated beings, disputed sentience, unknown ecosystems, emergent collectives, nonhuman intelligence, and scopes that may be invisible to current designation systems.

## 5. Domain quota discipline

For each domain of 100, require:

```trace
quota :=
  20 vulnerable_or_dependent_scopes
  + 15 constrained_actors
  + 15 high_power_actors
  + 10 hidden_labor_or_hidden_support_scopes
  + 10 ecological_or_system_scopes
  + 10 witness_or_truth_positions
  + 10 artificial_data_or_infrastructure_positions
  + 10 unknown_disputed_or_future_scopes
```

Quotas are not moral balance. They are anti-blind-spot pressure.

## 7. C/D/H discipline

For every row:

```trace
C := descriptive_contraction
D := designation_status
H := harm_reading_after_designation
```

Rules:

```trace
C_seen != H_resolved
D_unknown != clearance
D_excluded_requires_provenance
D_by_power_requires_contamination_note
H_reading_requires_clocks_and_designation
```

## 8. Standing / husbandry discipline

For every row where a scope is managed, preserved, protected, contained, trained, cared for, farmed, governed, moderated, or optimised:

```trace
standing_test := can_the_managed_party_alter_the_management_frame?
```

Outputs:

```trace
standing_status :=
  no_channel
  | data_only
  | appeal_within_frame
  | forced_routing_to_constraint_layer
  | D_layer_write_access
  | unknown
```

Boundary:

```trace
useful_preservation != standing
standing != care_certification
```

## 9. Failure-surface discipline

Each row should nominate one primary S-surface and one secondary S-surface where relevant.

```trace
S_surfaces :=
  S1_silent_contraction
  | S2_spoofed_viability
  | S3_endogenous_hardening
  | S4_correction_epistemic_cost_asymmetry
  | S5_scaffold_cage_reflexivity
```

If none applies, output:

```trace
COMPRESSION_ONLY_or_UNKNOWN
```

Do not force a surface.

## 10. Entity-handshake discipline

For each batch, identify where entity-to-entity handshake language is useful and where it fails.

```trace
handshake_test :=
  leverage
  + opacity
  + correction_handle
  + write_isolated_record
  + unknown_scope_protocol
  + non_self_certification
```

Key question:

```trace
correction_access_owed proportional_to leverage_times_opacity?
```

If this produces only a slogan, mark `COMPRESSION_ONLY`.

## 11. Sampling rule

Do not generate 1000 rows as a flat list.

Generate in ten batches. After each batch:

```trace
batch_review :=
  duplicate_detection
  + missing_scope_detection
  + bias_check
  + overclaim_check
  + new_pattern_candidates
  + demotion_candidates
```

Stop if the atlas becomes repetitive. Repetition is evidence.

## 13. First batch recommendation

Begin with Domain 10, not Domain 1.

Reason:

```trace
unknown_future_synthetic_scopes_pressure_the_designation_layer_hardest
```

If the atlas starts with familiar humans and animals, it may inherit familiar moral categories. Starting with unknown and synthetic scopes forces the C/D/H split to work before comfort arrives.
