# TRACE 100-Entity Middle-Out Atlas v0.1

Status: exploratory atlas / pressure surface. Not canon. Not validation. Not policy advice. Not a ranked moral list.

Purpose: test whether the current TRACE packet can middle-out across radically different entities and situations without collapsing into one scale, one ideology, one institution, or one kind of protected subject.

```trace
atlas_goal :=
  apply_middle_out_start
  across_100_entities
  → test_scope_clocks_claims_transitions_residue
```

Core question for every entry:

```trace
Given this entity or position,
what bounded encounter is live,
which scopes are affected,
which clocks are running,
which claims need typing,
which transitions can harden,
and what residue must not be erased?
```

---

## Method

Each entry uses a compact middle-out row:

```trace
Entry := {
  entity_or_position,
  first_scope_signal,
  active_clocks,
  laundering_or_dirty_edge,
  middle_out_move
}
```

No row is a complete answer. Each row is a starting discipline.
