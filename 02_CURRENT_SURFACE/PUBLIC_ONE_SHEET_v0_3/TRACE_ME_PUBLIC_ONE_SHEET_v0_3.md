Date: 2026-06-16

## What TRACE Is

TRACE is a diagnostic language for asking whether correction can reach affected subjects before harm hardens.

```trace
TRACE_core :=
  correction_before_hardening
  + K_gate
  + exit_when_correction_channel_is_harm_carrier
  + wrongability
```

This is narrower than earlier public surfaces. Recursive agency, creator responsibility, and contested legitimacy are preserved as candidate/teaching material, not active public spine.

## Core Test

```trace
live_correction := tau_correct < tau_harden
late_correction := tau_correct >= tau_harden
```

A safeguard is not real merely because it exists on paper. It must be able to bite before harm hardens, and the safeguard itself must remain open to correction.

## K-Gate

```trace
K := Witness * Record * Authority * Time * Enforcement * Repair
```

| Gate | Plain question |
|---|---|
| Witness | Can the harm and affected subject be seen? |
| Record | Is there a trustworthy, contestable record? |
| Authority | Can someone actually change the path? |
| Time | Can action arrive before hardening? |
| Enforcement | Will correction bind the system? |
| Repair | Can repair matter to the affected subject? |
