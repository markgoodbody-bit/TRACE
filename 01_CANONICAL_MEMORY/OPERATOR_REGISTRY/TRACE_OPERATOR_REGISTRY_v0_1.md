# TRACE Operator Registry v0.1

Date: 2026-06-16  
Status: canonical memory spine draft; not validation; not final theory.

## Purpose

The project has been discovering structures faster than the memory system can safely hold them.

This registry makes **operators** the main memory unit.

```trace
operator :=
  name
  + compression
  + definition
  + source_case
  + activation_condition
  + deactivation_condition
  + comparator
  + distinct_remainder
  + falsifier
  + demoter
  + status
  + must_not_claim
```

Cases teach operators. Claims status operators. Tests pressure operators. Reviews attack operators.

### OP-001 — Correction Before Hardening

**Family:** Timing  
**Status:** `core_operator`  
**Kernel status:** `in Kernel`  

```trace
live_correction := tau_correct < tau_harden
```

**Definition:** Correction must arrive before harm hardens into irreversible or functionally irreversible loss.

**Source case:** Apollo 13  
**Connected cases:** Columbia, Robodebt, Horizon, Public One-Sheet

**Activation condition:** Use when harm has a clock and correction has a delay.

**Deactivation condition:** Do not use when no action window can be defined without false precision.

**Comparator:** time-critical intervention; golden hour; incident response

**Distinct remainder:** Makes correction timing a central ethical diagnostic rather than an afterthought.

**Falsifier:** If clocks cannot be defined prospectively, demote to interpretive framing.

**Demoter:** If it cannot discriminate before outcome knowledge, demote in that domain.

**Must not claim:** Do not claim every delay is harm.
