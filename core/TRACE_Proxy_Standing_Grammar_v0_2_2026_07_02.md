## 0. Spine

```trace
consent_never_transfers_to_proxy
proxies_can_carry_objection_never_consent
```

Proxy standing is not representation that can authorise action. It is a provisional brake, bell, and record for scopes that cannot presently appear, object, or alter the frame directly.

```trace
proxy := brake + bell + record
proxy != signature
```

## 1. Definition

```trace
proxy_standing_v0_2 :=
  provisional_channel_for(absent | voiceless | unknown | future | dead | disputed_scope)
  + duty_direction(toward_scope, not_toward_process_completion)
  + powers(object + advocate_on_both_clocks + delay_before_T_irr
           + raise_D_omission + create_contestable_record + escalate)
  - powers(consent | clearance | close_D | discharge_residue)
  + forced_routing_to(constraint_layer + designation_layer)
  + interpolation_gap_declared
  + proxy_contamination_record(selection + payment + scope_setting + incentives)
  + subordination(any_direct_voice_outranks_proxy_immediately_and_prospectively)
  + obsolescence_duty(build_toward_direct_standing_where_theoretically_possible)
  + expiry_and_recheck(clocked_at_creation)
  - care_certification
  - final_designation
  - substitution_for_the_party
```

Plain English:

A proxy may object, urge, delay, force a question open, create a record, and escalate. It may not consent, clear, settle who counts, or close the books on loss. It must declare what it cannot know about the scope. It is outranked if direct standing becomes possible. Where direct standing could ever exist, the proxy must work toward its own retirement.

## 2. Duty direction

```trace
duty_direction := toward_scope_not_process_completion
```

A proxy whose loyalty is to completing the process is not a standing proxy. It is an administrative instrument.

## 3. Proxy powers

Allowed powers:

```trace
proxy_powers_allowed :=
  object
  + advocate_on_both_clocks
  + delay_before_T_irr
  + raise_D_omission
  + create_contestable_record
  + escalate
```

Forbidden powers:

```trace
proxy_powers_never_include :=
  consent
  + clearance
  + close_D
  + discharge_residue
```

The proxy must advocate on both loss and opportunity clocks. Pure delay can itself destroy futures.

## 7. Consent-theatre demoter

```trace
consent_theatre_demoter_v0_2 :=
  response_with_actual_frame_access(F)
  counted_or_presented_at_level(L)
  where L > F
  -> CONTAMINATED_STANDING_SIGNAL(label_inflation)
```

Access ladder:

```trace
feedback := response -> information_intake
```

Consent requires refusal possible, refusal affordable, and open frame.

## 8. Proxy clocks

```trace
T_proxy_select + T_proxy_route < T_irr
```

If not:

```trace
proxy_after_hardening := mourner_not_carrier
```

Required clocks:

```trace
T_proxy_expire := scheduled_at_creation
T_recontact := active_duty_clock_where_scope_may_become_contactable
T_reversal := reversal_path_has_own_validity_clock + must_be_periodically_exercised
T_D_reopen := scheduled + on_any_new_signal_from_or_about_scope
```

Override:

```trace
direct_standing_outranks_proxy(prospectively := immediately_and_unconditionally)
retrospectively := party_may_contest_all_proxy_era_decisions
```

If reversal is impossible after rejection:

```trace
action_reclassified := done_under_proxy_now_rejected
+ residue_of_rejected_proxy_action(permanent)
+ no_good_faith_discharge
```
