## 0. Why this note exists

Domain 10 of the 1000-existence atlas exposed a real hole:

```trace
standing_test_presupposes_present_expressive_party
```

The standing test asks:

```trace
can_the_managed_party_alter_the_management_frame?
```

But future, absent, voiceless, dead, simulated, unknown, or not-yet-visible scopes may not be able to answer, object, or alter any frame.

Without a companion rule, absence defaults to husbandry.

```trace
absence_is_the_husbandry_default
```

## 1. Designation-by-omission

Candidate pattern:

```trace
designation_by_omission :=
  scope_never_enters_frame
  + C_proceeds_without_D_record
  + no_H_reading_generated
```

This is not mere ignorance. It is an unrecorded designation effect.

Boundary:

```trace
not_named != not_affected
not_designated != unprotected
unimagined != unreal
```

Required row flag:

```trace
D_OMISSION_FLAG
```

Use when a scope may be affected but is absent from the designation lattice.

## 2. Proxy standing

Proxy standing is not standing. It is a weaker substitute used when direct standing is impossible or unavailable.

```trace
proxy_standing :=
  proxy_channel_for_absent_or_voiceless_or_unknown_scope
  + forced_routing_to_constraint_or_designation_layer
  + proxy_contamination_record
  + expiry_or_recheck
  - care_certification
  - final_designation
```

Proxy standing exists only if the proxy can alter the frame, not merely speak within it.

```trace
proxy_speech_without_frame_effect := data_only
```

## 3. Proxy contamination

Every proxy has capture risk.

```trace
proxy_contamination_sources :=
  selected_by_actor
  + paid_by_actor
  + scoped_by_actor
  + no_access_to_absent_scope
  + incentive_to_close_D
  + interest_in_managed_outcome
```

Proxy standing must carry contamination metadata.

```trace
proxy_claim != affected_party_claim
proxy_channel != standing_certification
```

## 4. Proxy standing ladder

```trace
proxy_standing_status :=
  no_proxy
  | actor_controlled_proxy
  | independent_proxy_no_write_access
  | independent_proxy_with_forced_routing
  | plural_proxy_with_conflict_record
  | proxy_plus_later_recontact_or_reversal_path
  | unknown
```

Higher ladder position is not moral clearance. It only records more structure.

## 5. Consent-theatre demoter

Candidate demoter:

```trace
solicited_then_nullified_response :=
  response_requested
  + response_logged_or_scored
  + response_cannot_change_frame
  -> CONTAMINATED_STANDING_SIGNAL
```

Plain English:

Asking for a response and then ensuring it cannot affect the management frame is worse than silence. It manufactures false standing.

Boundary:

```trace
asked != heard
logged != routed
preference_signal != standing
consent_theatre != consent
```

## 6. Atlas as interested estimator

The atlas itself assigns D-values. Therefore the atlas is an interested estimator unless proven otherwise.

```trace
atlas_D_assignment := estimator_output_under_contamination
```

The atlas cannot audit its own D-assignments.

```trace
atlas_self_audit_of_D := insufficient
```

Every batch should carry:

```trace
D_values_are_candidates_not_rulings
row_generation_is_designation_pressure_not_designation_authority
```


## 8. Handshake limit

Domain 10 exposed that the entity-handshake is a peer protocol.

```trace
handshake_requires := mutual_aperture + response_capacity + correction_handle
```

Non-peer cases include:

```trace
no_shared_aperture
+ one_way_aperture
+ inverting_aperture
```

Where mutual aperture is absent, the handshake degrades to unilateral restraint.

```trace
one_sided_handshake := husbandry_risk
```

Required output:

```trace
HANDSHAKE_LIMIT_FLAG
```
