## 1. Gradient as loop taxonomy

The gradient is real, but not a ladder. It is a taxonomy of loops that can coexist inside one system.

```trace
causation := transitions_without_outcome_attached_disposition
reactive := fixed_input_output_mapping
selection := mapping_shaped_by_outcomes
proto_responsibility := selection_whose_feedback_routes_through_other_selectors_responses
full_responsibility := selection_reachable_by_addressable_exchange_from_affected
reflective_responsibility := routing_internally_simulated_before_action
```

What changes at each transition:

```trace
where_outcome_feedback_attaches
+ who_can_route_into_it
```

## 4. Consequence gap

```trace
consequence_gap :=
  per_scope_divergence(
    influence_field(e) := leverage_over_futures_of(e),
    feedback_field(e) := capacity_of(e)_outcomes_to_modify_selector_dispositions
  )
```

The common pathology is not zero feedback but loop-swap:

```trace
loop_swap := influence_loop_and_consequence_bearing_loop_are_different_loops
```

Examples:

```trace

model_responds_to_rater_satisfaction_not_downstream_affected_party

```
