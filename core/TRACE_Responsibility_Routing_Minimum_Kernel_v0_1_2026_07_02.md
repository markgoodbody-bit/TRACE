## 1. Core claim

```trace
responsibility := per_scope_routing_table
not := property_of_agent_alone
```

Plain English:

Responsibility is not simply something an agent has. It is a relation between a selecting system and each affected scope: can that scope's outcomes route back into the selector's future disposition with enough force, speed, and depth to matter?

## 3. Routing-table form

```trace
responsibility(e, scope) :=
  route_from(scope_outcomes)
  to(e_future_dispositions)
```

## 4. Consequence gap

```trace
consequence_gap(e, scope) :=
  divergence(
    influence_field(e, scope),
    feedback_field(scope -> e_future_dispositions)
  )
```

Where:

```trace
influence_field := leverage_over_scope_future_space
feedback_field := capacity_of_scope_outcomes_to_modify_selector_disposition
```

## 5. Loop-swap

The common pathology is not no feedback. It is feedback attached to the wrong field.

