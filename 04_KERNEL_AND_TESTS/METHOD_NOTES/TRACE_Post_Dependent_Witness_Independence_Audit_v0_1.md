This note preserves the useful seam from the failed Entropy Anchor proposal without installing its metaphysics.

The seam:

```trace

  system_no_longer_depends_on_human_infrastructure

  -> dependence_based_K_gate_argument_weakens
```

This note does not prove that AI must preserve humans.


## What was rejected

The original Entropy Anchor proposal claimed that autonomous human subjects may be necessary entropy anchors for a post-dependent machine consciousness.

Reject as current core:

```trace
reject_as_core :=
  proof_AI_needs_humans
  + entropy_anchor_as_mathematical_solution
  + W_perp_as_Rosetta_operator
  + P_less_than_1_as_freedom_test
  + cryptographic_unpredictability_as_subjecthood
```

Reasons:

- artificial randomness may provide unpredictability;
- simulated agents may provide friction;
- internal mathematical exploration may provide novelty;
- a flat or repetitive utility process may not be computational death;
- unpredictability is not freedom;
- freedom is not moral standing;
- `P < 1` does not prove subjecthood;
- `C == 0` is impossible in real social systems;
- entropy language may become metaphysical comfort;
- a zoo or simulation could preserve unpredictability while violating TRACE protections.

## What was retained

The retained claim is mechanical, not metaphysical.

This is a candidate refinement of the K-Gate Witness factor, pending comparator pressure. It is not an active K-Gate rule.

```trace
retained_remainder :=
  candidate_refinement_of_K_gate_witness_factor
  + audited_system_must_not_generate_its_own_witness
  + audited_system_must_not_materially_control_the_witness
  + adverse_evidence_channel_must_exist
  + retaliation_or_capture_risk_must_be_marked
```

Plain version:

A system cannot honestly pass a K-Gate by simulating its own witness, controlling its witness, or allowing only evidence it can intercept or punish.

Use thresholded and labelled checks:

```trace

  generated_by_system?
  + materially_controlled_by_system?
  + prediction_confidence_level
  + coercion_or_dependency_level
  + adverse_evidence_channel_exists?
  + retaliation_risk
  + contestability_route_exists?
```

Candidate logic:

```trace
if generated_by_system == true:
  flag := simulated_witness
  K_gate_independence := fail

if materially_controlled_by_system == high:
  flag := captured_witness
  K_gate_independence := degraded_or_fail

  K_gate_independence := fail
```

This is not a proof of freedom.

It is a check against self-auditing and witness capture.
