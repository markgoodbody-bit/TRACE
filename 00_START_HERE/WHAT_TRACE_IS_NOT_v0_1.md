# What TRACE Is Not v0.1

Status: cold-reader claim-control file. Not canon. Not validation. This file exists to prevent overreading.

```trace
WHAT_TRACE_IS_NOT_v0_1 :=
  claim_boundary
  + reader_trust_guard
  + overclaim_prevention
  - proof
  - validation
```

## TRACE is not a finished ethics theory

TRACE is a working reasoning discipline and pattern language.

```trace
TRACE != finished_ethics_theory
```

It does not settle every moral conflict. It helps expose structure: harm, timing, option-space, burden, repair, and future-space.

## TRACE is not validated

TRACE has promising internal tests, hostile reviews, worked examples, and real-use signals.

That is not validation.

```trace
worked_example != validation
AI_review_agreement != validation
internal_coherence != validation
```

A claim becomes stronger only through disciplined pressure, comparison, use, error capture, and demotion.

## TRACE is not a replacement for expertise

TRACE can help ask better questions. It does not replace a doctor, vet, engineer, lawyer, domain expert, regulator, or person with local operational knowledge.

```trace
TRACE != expert_substitute
```

If a case needs expertise, TRACE should identify the right carrier, not impersonate it.

```trace
proper_use :=
  identify_expert_carrier
  + preserve_question_quality
  + clarify_risk_and_timing
```

## TRACE is not proof of AI consciousness or AI rights

TRACE can map AI systems as selection structures when they choose outputs, tools, or actions under constraints.

That does not prove inner experience, personhood, moral standing, or rights.

```trace
AI_selection_map != AI_personhood_proof
```

Do not turn functional mapping into metaphysical certainty.

## TRACE is not proof of free will

TRACE can map agency as selection under constraint.

It does not solve metaphysical free will.

```trace
agency_map != metaphysical_free_will_proof
```

The practical question is usually narrower:

```trace
practical_question :=
  what_selection_occurred?
  + who_shaped_options?
  + what_control_existed?
  + what_consequence_followed?
  + what_update_is_now_required?
```

## TRACE is not anti-system

TRACE does not assume institutions, rules, constraints, AI systems, or authority are bad.

```trace
constraint != oppression_always
institution != harm_always
AI_tool != danger_always
```

Good systems can preserve life, repair, dignity, and future-space.

Bad systems hide harm, delay correction, route burden downward, and make repair theatrical.

## TRACE is not vibes

TRACE is not just “this feels wrong.”

A TRACE read should name structure.

```trace
valid_read_needs :=
  affected_subject
  + harm_or_loss
  + clock
  + carrier
  + option_space
  + demoter
```

If it cannot name those, it may only be rhetoric.

## TRACE is not moral authority theatre

TRACE should not become a machine for declaring judgment from above.

```trace
TRACE != moral_authority_machine
```

It should make situations more answerable, not make the speaker feel more righteous.

## TRACE is not a licence to over-map everything

Some things are simple.

Some things are beautiful without needing TRACE.

Some things require direct action, not a larger conceptual map.

```trace
overmapping_failure :=
  elegant_structure
  + lost_life
  + no_better_action
```

If TRACE makes the case colder, vaguer, or less actionable, stop.

## TRACE is not a public claim of originality

TRACE may combine, compress, and reframe existing concerns: harm, correction, rights, responsibility, burden, time, institutions, ecology, agency, and AI alignment.

That does not mean every part is wholly original.

```trace
concordance_needed :=
  compare_existing_work
  + cite_sources
  + demote_novelty_claims
```

## TRACE is not the same as Mechanical Ethics

Mechanical Ethics is the broader project history and moral architecture.

TRACE is the current working pattern language and live-use surface.

```trace
Mechanical_Ethics := broader_project
TRACE := current_portable_reasoning_surface
```

Do not collapse them completely.

## TRACE is not finished

```trace
unfinished :=
  true
```

That is not a defect by itself. It is a claim boundary.

## Minimum honesty statement

```trace
minimum_honesty_statement :=
  TRACE_is_a_working_pattern_language
  + useful_for_structuring_hidden_harm_and_repair_questions
  - validated_solution
  - final_ethics_theory
```

Plain version:

TRACE may help you see where harm, timing, responsibility, and repair are being hidden. It does not prove itself right. Every TRACE reading needs a demoter.

End.
