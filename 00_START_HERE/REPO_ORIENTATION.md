## Repository discipline

No new bootstrap should be added unless it reveals, pressure-tests, or breaks a named operator.

No active claim should be immune from demotion.

No relay pack or one-sheet should be treated as canon.

## Blunt warning

If this repo becomes a museum of attractive fragments, it has failed.

If it lets TRACE lose claims, preserve payloads, and keep operators attackable, it is doing its job.
