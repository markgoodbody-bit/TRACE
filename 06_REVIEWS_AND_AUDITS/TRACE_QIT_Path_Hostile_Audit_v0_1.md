## 5. Example-specific attacks

### 5.1 Product vs entangled example

Useful: it blocks `composition_as_aggregation`.

Weakness: the separability discriminator `ad - bc` is only a simple pure two-qubit condition.

```trace
attack_product_entangled :=
  useful_for_basic_tensor_composition
  but:
    pure_two_qubit_only
    + not_general_entanglement_test
    + no_mixed_state_case
```

Patch:

```trace
patch_required :=
  explicitly_label_discriminator_as_pure_two_qubit_only
  + forbid_general_entanglement_inference
```

### 5.2 Measurement and record example

Useful: it blocks `record_equals_state`.

Weakness: it uses ideal computational-basis projective measurement only.

```trace
attack_measurement_record :=
  useful_for_record_state_distinction
  but:
    ideal_projective_measurement_only
    + no_general_POVM
    + no_noncommuting_sequence
```

Patch:

```trace
patch_required :=

  + forbid_general_measurement_theory_inference
```

### 5.3 Dephasing channel example

Useful: it blocks `loss_equals_forgetting`.
