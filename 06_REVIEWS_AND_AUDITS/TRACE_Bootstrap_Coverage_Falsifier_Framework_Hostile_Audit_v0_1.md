## 4. Top 5 drift risks

1. **Detector-vs-translation collapse** — if ordinary expert reasoning beats TRACE, the current patch may demote too much. TRACE can fail as detector but survive as interlingua.
2. **Falsifier theatre** — some falsifiers could become broad disclaimers unless tied to actual worked comparisons.
3. **Bootstrap pack status confusion** — the falsifier patch exists outside Bootstrap V2; future readers may think it is already installed.
4. **Coverage matrix freeze** — the matrix may become stale quickly as support patches are created.
5. **Universality overcorrection** — fear of overclaim may shrink TRACE from cross-domain communication into a small checklist.

## 5. Top 5 strengths that survive hostile reading

1. The matrix correctly blocks story collection and treats stories as carriers, not evidence.
2. The matrix names real blanks instead of pretending Bootstrap V2 is complete.
3. The falsifier patch correctly distinguishes misuse guards from inward falsifiers.
4. The cross-cluster expert-reasoning falsifier is directionally right: TRACE must lose if expert reasoning does better and TRACE adds no transfer value.
5. The false-universality falsifier is necessary and protects the project from becoming a vocabulary empire.
