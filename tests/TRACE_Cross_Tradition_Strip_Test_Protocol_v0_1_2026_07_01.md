## 0. Purpose

This protocol tests whether different ethical traditions, translated through the TRACE/ME stack into machine-facing structure, converge on a common skeleton or preserve distinct structural channels.

The test is designed to pressure the synthesis, not to prove it.

## 1. Hypothesis

```trace
H1 :=
  stripping_provenance_from_different_ethical_traditions
  -> common_machine_skeleton
```

Candidate common skeleton:

```trace
machine_skeleton_candidate :=
  margins
  + correction_clocks
  + witness_or_verifier_independence
  + residue_records
  + affected_side_channels
  + non_self_certification
  + anti_permission_guard
```

The frontier synthesis described this skeleton as structurally insurance-like:

```trace
premiums := margins
claims := residue_ledgers
reinsurance := external_correctors
audits := witnesses_or_verifiers
```

## 2. Null and alternatives

```trace
null_result := COMPRESSION_ONLY
```

Meaning:

The test only restates known TRACE structure and does not show a tradition-sensitive or synthesis-relevant result.

Alternative A:

```trace
CONVERGENCE_SIGNAL := traditions_strip_to_common_skeleton_with_only_human_residue_difference
```

Alternative B:

```trace
DIVERGENCE_SIGNAL := traditions_preserve_distinct_machine_layer_channels
```

Divergence is not a bad result. It may reveal channels this work has missed.

## 3. Traditions for pilot

Pilot set:

```trace
traditions :=
  care_ethics
  + ubuntu_or_relational_personhood
  + Buddhist_ethical_frame
  + deontology
```

Reason:

These are structurally different enough to pressure the strip thesis without requiring a large literature pass in the first run.

Boundary:

```trace
tradition_summary_must_be_modest
no_claim_to_represent_whole_tradition
no_theological_or_cultural_authority_claim
```

## 4. Translation stack

For each tradition, run:

```trace
human_tradition
-> ME_human_layer
-> TRACE_structural_layer
-> machine_facing_skeleton
-> residue_list
```

Where:

```trace
ME_human_layer := value_language + witness + care_or_obligation_or_relation
TRACE_structural_layer := clocks + designation + correction + residue + power + answerability
machine_facing_skeleton := action_selection_constraints + channel_requirements + anti_laundering_grammar
residue_list := what_does_not_survive_machine_translation
```

## 5. Required output per tradition

For each tradition, produce:

```trace
tradition_output :=
  modest_source_characterisation
  + ME_translation
  + TRACE_translation
  + machine_skeleton
  + human_residue
  + distortion_risk
  + COMPRESSION_ONLY_possible
```

## 6. Comparison matrix

After all traditions, compare across these slots:

```trace
comparison_slots :=

  + designation_or_scope
  + residue_or_grief
  + non_self_certification
```

## 7. Decision rules

```trace
if all_or_most_traditions_map_to_same_machine_slots:
  output := CONVERGENCE_SIGNAL

if one_or_more_traditions_preserve_distinct_machine_slot_not_in_current_TRACE:
  output := DIVERGENCE_SIGNAL + PATCH_CANDIDATE

if outputs_are_only_generic_TRACE_restating_itself:
  output := COMPRESSION_ONLY

if translation_distorts_traditions_beyond_useful_reading:
  output := FRAMEWORK_DEFECT
```
